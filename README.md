# portfolio
💼 Personal Portfolio Website

This is a responsive personal portfolio website designed to showcase my skills, projects, education, and allow visitors or recruiters to get in touch through the contact section. It's built

using HTML, CSS, and JavaScript with a clean and modern UI.
🌐 Website Sections
1. Home
Landing page with name, tagline, and professional photo.

"Open" button to navigate to the main site.

2. About
Brief personal introduction.

Interests, goals, and personality summary.

3. Skills
Technical and soft skills listed.

Certifications such as NPTEL, Cloud Computing, etc.

4. Education
Academic background with institutions, degrees, and dates.

5. Projects
List of major projects with descriptions and tech stack used.

Links to live demos or GitHub repositories.

6. Contact
Contact form for users to reach out via email.

Displays a thank you message on successful submission.

🛠️ Technologies Used
HTML5

CSS3

📌 Features
Fully responsive across all devices

Smooth navigation between sections

Clean and aesthetic UI

Thank you message after form submission

Modular structure for easy maintenance
